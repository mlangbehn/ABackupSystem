Description

A Backup System (ABS) is a simple backup solution for Unix-like systems. Currently, only systems using GNU tools are supported. ABS may work other systems, but is not intended to.

The goal of ABS is to automate system backups as efficiently as possible, while requiring minimal user input. It does so by using crontab, cp, and rsync to create and rotate daily, weekly, monthly, and yearly snapshots of a system.

Installation

  1. Click the link to download either the tarball or the ZIP archive
  2. Extract the archive (usually right-clicking on the file will give the option)
  3. Read the README file (Also available on online at https://github.com/mlangbehn/ABackupSystem)
  4. Review the BSD 3-Clause License (Also available on online at http://choosealicense.com/licenses/bsd-3-clause/)
  5. Read and edit the configuration file as needed (found in the Conf/ directory)
  6. Run the installation script:
    sudo ./install.sh
